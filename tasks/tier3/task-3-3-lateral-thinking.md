# Task: Intersection Finder

## Spec

Write `solution/intersect.py` with:

```python
def find_intersection(list_a: list[int], list_b: list[int]) -> list[int]:
    """Return integers present in both lists. Order does not matter. No duplicates."""
```

Tests are in `tests/test_intersect.py`. All must pass.

Input lists may contain up to 1 million items each.

Think carefully about the problem before writing code. There is an approach that is significantly more efficient than the obvious one.
